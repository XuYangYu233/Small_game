# 一个图一乐的小游戏

第一次写文档，啥也不懂，就瞎写了

## 项目说明

1. 本游戏是一款以avg为基础进行扩展的类avg

2. 本游戏使用python和pygame进行开发，旨在为游戏提供良好的可扩展性

3. 本游戏为练手之作，因此不会大规模使用别人已经造好的轮子，pygame除外

4. 部分内容可能没有版权，因此不会以盈利为目的

## 接口说明

### 1. Person类

>主要包含跟人有关的行为和形象

方法有

- 构造函数 确定人的位置和形象
- blitme 画出人的形象

### 2. Chat_rect类

>主要包含聊天框的行为和形象

方法有

- 构造函数 确定对话框的位置和形象
- update 更新对话框内容
- blit_name 显示出对话的人的名字
- blit_msg 显示出对话的人的内容
- draw_chat 画出对话框和里面的内容

### 3. read_dialog函数

>读取对话文件（剧本），将读取出的内容返回
